# House Price Prediction

End-to-end machine learning project to predict house sale prices.

## Results
| Model | RMSE |
|-------|------|
| Ridge Regression | 0.1594 |
| Random Forest | 0.1393 |
| XGBoost | 0.1288 ✓ Best |

## Tech Stack
Python, XGBoost, Scikit-learn, SHAP, Pandas, Matplotlib

## How to Run
1. Clone repo
2. pip install -r requirements.txt
3. Add train.csv to data/ folder
4. Open notebooks/house_price.ipynb
